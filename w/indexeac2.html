We are looking for mirrors

We are looking for mirror service. For the moment we provide only a single URL from which you could download ArchPwn. It would be great if someone with a mirror service would help us. If you have a similar service don't hesitate to contact us.

Mirrors

Official mirror

This is the official mirror. Please avoid to use this because the bandwitch is very limited. Try another mirror instead :)

  • HTTP official mirror

itsredegg.com

Thanks again to Mike Staszel and Firefly Linux for this mirror accessible via HTTP:

  • HTTP itsredegg.com mirror

garr mirror

Thanks to garr for providing us an http/ftp/rsync mirror:

  • HTTP garr.it mirror
  • FTP garr.it mirror

RSYNC access is available under: rsync://archpwn.mirror.garr.it/archpwn/

ibiblio mirror

Really thanks to ibiblio.org team that helps us providing an http/ftp/rsync mirror:

  • HTTP ibiblio.org mirror
  • FTP ibiblio.org mirror

RSYNC access is available under: rsync://distro.ibiblio.org::distros/archpwn/

File:Ibiblio hosted2-110x32.png

vim.org mirror

It seems that ftp.vim.org is synced with ibiblio.org so this mirror is also available (HTTP/FTP):

  • HTTP vim.org mirror
  • FTP vim.org mirror

nluug.nl

This is an unofficial mirror like the vim.org one. Access is available trough HTTP and FTP:

  • HTTP nluug.nl mirror
  • FTP nluug.nl mirror
