General

General questions.

What is ArchPwn?

ArchPwn is a security based Linux Live CD/DVD system that aims to be a fully working swiss army knife for pentesting purpose (GPL licensed). It's based on the famous ArchLinux distribution and inherits its KISS philosphy.

During the development process, we have tried to keep the original base system intact and to create on top of it a complete repository of security tools that every pentester may need. At the current state, the distribution provide a set of ca 300 tools.

Why do you choose ArchLinux as base?

Because Arch is speedy, fresh and clean. It's like a Slack but with a more emphasis to the software management. Moreover, it is easily adaptable to create new systems. And last but not least, we're familiar with Arch because we use it as every day distro and we love it :)

Why another security oriented Linux live system?

Essentialy, we are not happy about the current situation. The most known security oriented distribution are only suitable for a live sessions. With an external repository you can install the ArchPwn software suite on your ArchLinux installation, and turns your workstation in a pwn station!

A second reason that led us to start this huge project is due to the fact that other teams are closed to contributions. The ArchPwn team on the other hand, encourages everyone to join the development process. In fact, the entire source code of the project is hosted under github. You're free to follow our changes or fork us to make ArchPwn better!

Repository stuff

Program xyz doesn't work properly/sigsegv/etc ..

Please create a new issue with detailed information by using the Issue tracker.

Program xyz is not present in the repository tree

If you think that software xyz should be present in ArchPwn then create a the new issue asking for inclusion. Use request label.

If you're familiar with PKGBUILD format, it's preferable to include a draft in the body of the request. Use this model as reference.
